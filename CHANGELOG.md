# Changelog

All notable changes to this project will be documented in this file.

The format is based on Keep a Changelog and this project follows Semantic Versioning.

## [0.1.0] - 03/17/2026

### Added
- LaTeX resume, CV, and cover letter sources with modern styling.
- Build automation via Makefile targets for full and per-document PDF generation.
- GitHub Actions workflows for LaTeX build, release management, GitHub Pages publishing, and conventional commit validation.
- Resume and cover letter tailoring system with AI-assisted prompt suite.
- CV/resume strategy documentation (CV as truth source, resume tailored to opportunities).
- Work summaries folder with daily standup reports and archived notes.
- Comprehensive prompt library for resume scoring, job description analysis, and content tailoring.

### Changed
- Updated resume content with emphasis on platform engineering and SRE/SLO experience.
- Refreshed LaTeX styling for improved visual hierarchy and readability.
- Refined CI/CD workflows with targeted path filters for efficient builds.
- Enhanced cover letter formatting and tailoring capabilities.

### Fixed
- Resolved resume bullet point formatting and spacing issues.
- Corrected time format consistency across CV entries.
- Fixed GitHub Pages publishing flow and artifact reuse.
