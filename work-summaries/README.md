# Work Summaries

This directory contains detailed summaries of projects and work experiences that can be used as reference material for resume customization, cover letters, and AI prompt context.

## Structure

Each file should contain:
- **Company/Project Name**
- **Role & Duration**
- **Technologies Used**
- **Detailed Description of Work**
- **Key Achievements**
- **Challenges & Solutions**
- **Technical Details** (architecture, patterns, etc.)

## Files

- `superior-tech-research-2025-backend.md` - Backend development for HR performance platform (Go, Kubernetes, SQL Server)
- `superior-tech-research-2025-frontend.md` - Frontend development for HR performance platform (Vue.js 3, TypeScript, Vuetify)

Additional summaries can be added as separate markdown files following the naming convention: `company-name-year-project-description.md`
